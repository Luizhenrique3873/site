Evolução do HTML

  • HTML 1

    • O HTML original, conforme criado por Tim Berners-Lee, não foi formalmente versionado como "HTML 1". Ele era uma linguagem bastante básica e simples, projetada para estruturar documentos hipertexto e criar links entre eles.

  • HTML 2

    • O HTML 2 foi publicado em 1995, pouco depois do lançamento do HTML original, para melhorar e expandir as capacidades da linguagem,o HTML 2 introduziu várias novas tags e recursos em relação ao HTML original

  • HTML 3

    • Uma das características notáveis que começou a se desenvolver nessa época foi o suporte a tabelas mais complexas. Isso permitiu layouts mais avançados, embora as técnicas usadas fossem muitas vezes complicadas e não padronizadas entre diferentes navegadores,os formulários também começaram a ser mais sofisticados, com a adição de recursos como campos de entrada de arquivo, botões de rádio e caixas de seleção.

  • HTML 4

    • O HTML 4 foi lançado em 1997 pelo World Wide Web Consortium (W3C) como uma evolução significativa em relação ao HTML 3.2 e para enfrentar os desafios crescentes da criação de conteúdo web. O HTML 4 trouxe melhorias substanciais nos formulários da web, introduzindo elementos como label, fieldset, legend, button e optgroup, tornando a criação e estilização de formulários mais eficiente e semântica. O HTML 4 trabalhou em conjunto com as folhas de estilo em cascata (CSS) para permitir uma formatação mais avançada e consistente das páginas da web. O CSS separou a estrutura do conteúdo da sua apresentação, o que levou a uma maior flexibilidade de design.

  • HTML 5

    • O HTML5 é a quinta versão do HyperText Markup Language, a linguagem de marcação utilizada para criar páginas web. Foi desenvolvido com o objetivo de modernizar e aprimorar as capacidades da web, oferecendo uma variedade de recursos novos e avançados.

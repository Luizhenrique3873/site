Título Principal

Subtítulo 1

Este é um parágrafo de texto que demonstra o uso da tag <p>. Um parágrafo é usado para separar o conteúdo em blocos distintos.

Subtítulo 2

Outro parágrafo de texto.
A tag <br> é usada para criar uma quebra de linha dentro do mesmo parágrafo.

Subtítulo 3

Mais um parágrafo de texto. Você também pode usar a tag <br> para criar uma quebra de linha.
Além disso, a tag <hr> pode ser usada para criar uma linha horizontal para separar o conteúdo.

Subtítulo 4

Aqui estamos usando diferentes níveis de títulos, de <h1> a <h6>. Eles têm tamanhos e importâncias diferentes.

Subtítulo 5

Os títulos também ajudam a organizar o conteúdo de maneira hierárquica.

Subtítulo 6

Este é o menor nível de título. É usado para detalhes menores ou informações de apoio.

Neste exemplo tivemos: O uso de títulos de <h1> até <h6> para criar diferentes níveis de cabeçalhos. A tag <p> para criar parágrafos de texto. A tag <br> para criar quebras de linha dentro de um parágrafo. A tag <hr> para criar uma linha horizontal que separa o conteúdo.

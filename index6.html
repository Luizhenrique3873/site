Exemplo de Uso da Tag Lista





Lista de Compras

Aqui está uma lista de compras usando a tag <ul>:

  • Maçãs
  • Bananas
  • Laranjas
  • Morango

Nesse exemplo, usamos a tag <ul> para criar uma lista não ordenada e a tag <li> para cada item da lista. Isso resulta em uma lista de compras com quatro itens. O navegador irá renderizar essa lista com marcadores padrão (normalmente pontos pretos) antes de cada item.
